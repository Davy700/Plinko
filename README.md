# Plinko
This program is a Python implementation of Plinko.

# How to play
At the start of the game $100 virtual money will be given to the user. After pressing SPACE a ball holding the value of the bet will appear that will fall down. If the ball collides with a brick, you will receive a payout equal to the multiplier of the brick where the ball lands. At the upper-left corner the adjustable bet, at the upper-right corner the current money is displayed.

# Controls
- SPACE - starting a ball
- Up - increasing the bet by $0.25
- Down - reducing the bet by $0.25
